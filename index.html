PROJECTS

Graph Component Counter

This is a small project I worked on to practice implementing an adjacency matrix and breadth-first search.

My main method accepts an argument for a text file whose first line specifies the number of nodes |V| and whose suceeding lines contain two integers indicating an edge between the two nodes. Nodes in the graph are numbered sequentially from 0..|V| and when run, the program will print the number of connected components in the graph.




Cancer Classifier

This was the final assignment for my CSCI 141 - Intro to Programming class (Winter 2020). The goal was to implement a Machine Learning framework using Python to develop a rule-based classifier that predicts if an unknown tumor is malignant or benign.
